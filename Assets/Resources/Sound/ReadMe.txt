[SE 용도 설명]
------------

- 기본 -
Click: 기본 클릭음
Buzzer: 클릭이 불가능한 걸 클릭했을 때 나는 클릭음
------------------------------------------

- 블록 퍼즐 플레이 -
Block_Click: 블록 클릭 시에만 나는 클릭음
Merge_Block: 블록 머지 효과음

Hit_01
Hit_02: 피격음. (PC나 적이 1회 피격당할 때마다 (1.) 1회씩 재생되며, (2.) 둘 중 한 개의 SE가 50%의 확률로 랜덤하게 재생됨)

Item: 아이템 사용음

Special_01
Special_02: 필살기 효과음. 두 개가 함께 재생됨

Death: 전투 불능 효과음

Win: Win 문구와 함께 재생됨
Lose: Lose 문구와 함께 재생됨
Start: Start 문구와 함께 재생됨
------------------------------------------

-미연시-
Foot_Step: 후다닥 달려가는 소리에 사용됨. 
	  (한 번 사용할 때마다: 0.2초마다 1회씩, 총 5회를 재생함 / 0.2sec x 5.0 == 1.0sec )
------------------------------------------
